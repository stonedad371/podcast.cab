核心能力

🎬

一拖即出片

拖入 mp3 + srt，自动跑完：切章节 → 写标题副标题 → 挑金句 → 生封面 → 每章生背景图 → 渲染 mp4。可设"全部就绪后自动渲染"。

🖼️

每章独立 AI 背景图，可重生

LLM 按每章节内容写英文提示词，MiniMax 出图。**不满意点 🔄 重生**：填一句中文提示（如"画得更暗一点"、"用瀑布隐喻"），LLM 综合产新 prompt 重出。

📝

顺手写好发布文案

分析时一并产出**短视频平台**用的标题（带钩子）、简介正文、5–10 个话题标签。一键复制到抖音 / 小红书 / 视频号 直接贴。

🎨

AI 生 9:16 封面

视频首帧即封面合成（背景图 + 标题 + 副标题 + 品牌条 + accent 线），上传平台抓首帧当缩略图。

💬

章节卡 + 金句卡 + 字幕同步

字幕实时跟声音对齐，章节切换弹胶囊标签，金句到点出大字卡。字幕跟音频对不齐？设置里 slider 微调时间补偿。

✨

视觉不单调

浮动粒子 + Ken Burns 多向平移 + 章节切换柔光 flash。每章独立"相机方向"，cross-fade 像换了视角。

👀

片尾互动引导

3 秒片尾自动出"关注 / 点赞 / 评论 / 收藏"4 个动效卡片，专为短视频平台优化的"涨粉"片尾。

🏷️

自定义品牌

设置里填你的品牌名，视频里所有 podcast.cab 字样替换成你的。封面、片头条、片中右上角都改。

🎬

在线预览 + 中文文件名

渲染完页面里直接 9:16 播放器播，不用先下载。要保存时文件名自动用你的中文标题，不是 UUID。

🔒

完全本地 · 一个 API key

Docker 跑在你机器上，音频/字幕/key/成品都在本机，podcast.cab 不收任何数据。一个 MiniMax key 同时跑文本分析 + 生图。

⚛️

Remotion 渲染，全开源

React 渲染每一帧，1080×1920 @ 30fps H.264 + AAC。配色字体布局都在 remotion/Composition.tsx，会 React 就能改。MIT 许可。

三步出片

1

装 Docker Desktop

一次性安装，官网下载，或用 OrbStack（国内可下、更轻）。装完打开等鲸鱼图标变绿。

2

下载本工具 + 启动

从 GitHub clone 或下载 ZIP，Mac 双击 start.command，浏览器自动打开 localhost:3010。

3

拖入音频 + 字幕

填一次 MiniMax key，拖入 mp3 + srt。等几分钟看预览，不满意点重生章节图，满意点开始生成，下载 mp4 + 复制发布文案。

没有播客音频和字幕？ 用 yuban.yun 一键生成播客音频与时间戳字幕，导出 mp3 + srt 直接喂给 podcast.cab。

几个常见问题

章节图不满意怎么办？

每张章节图下方有 🔄 重生按钮。点开后弹文本框，可以填一句中文提示（如"画得更暗一点"、"用瀑布隐喻"），LLM 会综合你的提示 + 章节字幕节选重写英文 prompt，再调 MiniMax 出图。不写提示也可以纯抽一次重新生成。

字幕跟声音对不齐怎么办？

设置里有"字幕时间补偿" slider，范围 -1.0 到 +1.0 秒，0.05 步进。字幕比声音早出现就往右拖（延后字幕），反之往左拖。默认 +0.2s 补偿 ASR 工具普遍的字幕提前现象。预览实时反映，不用重新渲染。

发布文案是什么？跟视频里的标题一样吗？

不一样。视频里的标题（如"8 年亏货到稳定盈利：交易者的至暗与翻盘"）讲究信息量；发布文案的标题（platformTitle）讲究**抓点击**，可以更短更钩子，可带 1-2 个 emoji。还附送一段 50-200 字简介 + 5-10 个话题标签，分别有复制按钮，一键贴到抖音 / 小红书 / 视频号。

需要联网吗？

需要——LLM 切章节 / 写标题 / 生封面 / 生章节图 / 写发布文案 都要打 MiniMax 的 API。视频本身的 Remotion 渲染不需要联网。

API key 安全吗？数据会上传吗？

Key 存在你本地 ./data/config/api-keys.json，权限 600，不会进 Docker 镜像、不会上传任何地方。podcast.cab 也不收集任何数据——你的所有素材和成品都在你机器上。

渲染要多久？

首次约 15–30 分钟（要下 ~93MB Chromium 镜像）。之后流程：分析 15-40s + 封面 15-30s + 5 张章节图 1-2min + 渲染 3-5min ≈ 5-8 分钟出一条 4 分钟视频。机器配置越好越快。

支持横版 16:9 吗？

V1 只输出 9:16 竖版（抖音 / 小红书 / 视频号 / B 站竖屏 原生）。横版 16:9 在路上。

能改样式吗？

全开源。配色 / 字体 / 布局 / 动效都在 remotion/Composition.tsx，会 React 就能改。粒子数量、Ken Burns 方向、章节切换 flash 颜色这些常用变量都在文件顶部常量区。
